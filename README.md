A collection of graphical programming experiments, simulations, and visual projects built in both Java and Python. This repository focuses on interactive graphics, recursion, simulations, and algorithmic visualization while exploring GUI design and user interaction.

# Projects Included

## 1. Fractal Tree (Java)

### Files

* `FractalTree.java`
* `AsymmetricalFractalTree.java`

### Description

A recursive fractal tree visualization built using Java Swing and `JFrame`. The program dynamically generates branching tree structures using recursive geometry and adjustable branch angles.

### Features

* Recursive branch generation
* Keyboard-controlled branch angle adjustments
* Interactive tree rendering
* Swing-based graphical interface
* Black-and-white minimalist visualization style
* Symmetrical and asymmetrical tree variants

### Concepts Demonstrated

* Recursion
* Java Swing GUI development
* Graphics rendering with `Graphics2D`
* Event handling
* Trigonometry and coordinate calculations
* Interactive visualization

### Running the Java Version

Compile and run using:

```bash
javac FractalTree.java
java FractalTree
```

For the asymmetrical version:

```bash
javac AsymmetricalFractalTree.java
java AsymmetricalFractalTree
```

## 2. Fractal Tree (Python)

### File

### Description

A Python implementation of a recursive fractal tree generator. This version explores recursive rendering and graphical drawing using Python graphics libraries.

### Features

* Recursive tree generation
* Adjustable branch depth and angles
* Real-time graphical rendering
* Lightweight simulation structure

### Concepts Demonstrated

* Python recursion
* Mathematical visualization

### Running the Python Version

```bash
pip install pygame
```


## 3. Conway's Game of Life (Python)

### File

* `GameOfLife.py`

### Description

An implementation of Conway’s Game of Life, a famous cellular automaton simulation where cells evolve over time based on neighboring states.

### Features

* Interactive cellular simulation
* Real-time generation updates
* Grid-based rendering
* Classic Conway rule implementation
* Pause and reset functionality

### Concepts Demonstrated

* Cellular automata
* Simulation programming
* Grid-based algorithms
* State management
* Python graphical rendering

### Running the Project

Install dependencies if necessary:

```bash
pip install pygame
```

Run the project:

```bash
python GameOfLife.py
```
